# 花呗与借呗用户借款行为诊断分析

## 项目简介
基于花呗与借呗2020年4-7月真实业务数据，从风险管理和业务增长双重视角，对两大信贷产品进行全面诊断分析。

## 核心发现
- 借呗占成交额99.9%，风险高度集中于12期、18期等长期限产品
- 花呗6期以9.32%客户贡献46.27%金额，存在套现风险信号
- 业绩呈“沿海 > 内陆”格局，同城各业务组产品结构高度同质化
- 人员配比与人均效率无显著关联，市场体量是效率的首要决定因素

## 分析框架
1. 产品规模与风险基调
2. 借款期限结构风险
3. 时间趋势分析
4. 地域与团队效能
5. 人员配置效率诊断
